# 🎂 Age Calculator

A responsive Age Calculator built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features

- **Date of Birth Input** – Select your DOB using a date picker
- **Calculate Age** – Displays exact age in Years, Months, and Days
- **Result Card** – Beautiful card layout showing age breakdown
- **Validation** – Handles empty input and future dates gracefully
- **Dark / Light Mode** – Toggle themes; preference saved in Local Storage
- **Reset Button** – Clears all inputs and results without page refresh
- **Live Today's Date** – Shows current date at the top
- **Next Birthday Countdown** – Days remaining until next birthday
- **Total Days Lived** – Count of days from birth to today
- **Age Category** – Classifies as Child / Teen / Adult / Senior
- **Birthday Wish** – Special message if today is your birthday
- **Responsive Design** – Works on desktop, tablet, and mobile

## How to Use

1. Open `index.html` in any browser.
2. Select your Date of Birth from the date picker.
3. Click **Calculate Age** to see your age.
4. Click **Reset** to clear everything.
5. Use the **🌙 / ☀️** toggle to switch themes.
