# Modern Calculator

A responsive, feature-rich calculator web application built with **HTML, CSS, and Vanilla JavaScript**.

## Features

- Basic arithmetic operations (+, -, ×, ÷, %)
- Decimal numbers and bracket support
- Keyboard support (0-9, +, -, *, /, ., Enter, Backspace, Escape)
- Dark Mode / Light Mode toggle (saved in Local Storage)
- Calculation history with clear option (saved in Local Storage)
- Responsive design (works on desktop, tablet, and mobile)
- Button hover and click animations
- Error handling (invalid expressions, division by zero)

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Enter numbers |
| `+ - * /` | Operators |
| `.` | Decimal point |
| `Enter` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` | Clear all |

## How to Run

Open `index.html` in any modern web browser.
